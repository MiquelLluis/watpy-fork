# Classes for managing the CoRe DB

Database structure

     <path_db>/
     | --- <code>_<simulation_number>/
     |     | --- R??/
     | 	 | 	 | --- data.h5
     |	 | 	 | --- metadata.txt 
     ...   
     | 	 | --- metadata_main.txt

The `database_key` is composed as `<code>:<simulation_number>:R<run_number>`.
